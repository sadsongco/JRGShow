Class: VisualiserTemplate

VisualiserTemplate()

Dummy class to act as documentation / guide on building a visualiser. Class name must match the filename exactly, minus the .js filename extension. To activate it, register the class in registeredVis.js. Remove this comment when you create your new visualiser!
